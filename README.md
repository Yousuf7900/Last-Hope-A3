# Football Ticket Booking System

## Overview

This project is a simple Football Ticket Booking System database built using PostgreSQL.

The system manages:

* Users
* Football Matches
* Ticket Bookings

It demonstrates database design concepts including primary keys, foreign keys, constraints, joins, subqueries, and data filtering.

## Database Tables

### Users

Stores football fan and ticket manager information.

### Matches

Stores football match details, ticket prices, and match availability status.

### Bookings

Stores ticket booking information and connects users with matches.

## Features

* User Management
* Match Management
* Ticket Booking Records
* Booking Payment Status Tracking
* Match Availability Status Tracking

## SQL Concepts Used

* Primary Key
* Foreign Key
* UNIQUE Constraint
* CHECK Constraint
* INNER JOIN
* LEFT JOIN
* Subquery
* COALESCE
* ORDER BY
* LIMIT & OFFSET
